• Title
    Appendix to “Expanded Algal Cultivation Can Reverse Planetary Boundary Transgressions”

  • Introduction
    This appendix contains a description of the spreadsheet EACCRPBT.xlsx, codebooks for the datasets computed by Precompute.R, a discussion of possible future directions for improving the accuracy and comprehensiveness of this work, and a complete listing (Table A1) of prioritized basins excerpted in Table 2 of the manuscript.

  • EACCRPBT.xlsx
    EACCRPBT.xlsx provides a user-friendly interface for determining parameters to be used in the economic model described in the manuscript. Various assumptions and conversion factors are used to refine the input parameters to the model, which runs by loading the variables directly from the workbook “R Variables”.
